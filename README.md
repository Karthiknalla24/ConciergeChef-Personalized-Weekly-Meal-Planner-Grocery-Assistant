## Overview
ConciergeChef is a Concierge Agents capstone project that automates weekly dinner planning, produces an optimized shopping list, and schedules cooking reminders. It demonstrates multi-agent orchestration, tools usage, sessions & memory, and basic observability.

## Features
- Personalized weekly dinner plan (7 meals)
- Pantry-first logic to reduce waste
- Shopping list aggregation & estimated cost
- Optional calendar scheduling for meal reminders
- MemoryBank stores user preferences and last plan
